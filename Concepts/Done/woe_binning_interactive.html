Calculate distributions
Express each bin's events and non-events as % of the total events and non-events.
Dist. of Events (bin i) = Events_i / Total Events = Events_i / 250
Dist. of Non-Events (bin i) = Non-Events_i / Total Non-Events = Non-Events_i / 750
3

WoE = ln(Distribution of Events ÷ Distribution of Non-Events)
WoE_i = ln( (Events_i / 250) / (Non-Events_i / 750) )

Same 1,000 applicants, same Age feature — now using Equal-Frequency (Quantile) Binning into 4 quartiles.

1
Sort all values, split into 4 equal groups
Each bin has exactly 250 people. Bin edges come from the 25th, 50th, 75th percentiles.
n = 1000, k = 4 → each bin has 1000/4 = 250 rows

Q1 (25th pct) = Age 29 → Bin 1: Age 18–29
Q2 (50th pct) = Age 38 → Bin 2: Age 30–38
Q3 (75th pct) = Age 50 → Bin 3: Age 39–50
Bin 4: Age 51–75

Key observations
✓ All bins have equal count (250) — no empty or sparse bins
✓ Bins vary in width: Q4 spans 24 years vs Q2's 8 years (skew handled automatically)
✗ Bin edges (29, 38, 50) are arbitrary — no domain meaning
✗ Not informed by the target — bins aren't necessarily predictive

Same dataset — now a shallow decision tree learns the optimal split points from the data.

1
Fit a tree on one feature
Train DecisionTreeClassifier with max_leaf_nodes=4 on Age vs Default. The tree searches every possible split point and picks the one that minimises Gini impurity.
from sklearn.tree import DecisionTreeClassifier
tree = DecisionTreeClassifier(max_leaf_nodes=4, random_state=0)
tree.fit(X[['age']], y)

# Tree finds optimal splits at: 27.5, 42.5, 55.5
thresholds = tree.tree_.threshold # → [27.5, 42.5, 55.5]
